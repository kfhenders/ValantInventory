Valant Inventory Coding Excercise API

The following methods are exposed:

  1. [POST] /api/Inventory - Creates an item in the inventory
  2. [GET] /api/Inventory/{label} - Get's the item in the inventory with the requested label
  3. [PUT] /api/Inventory - Updates an exisitng item in the inventory or creates a new one if it doesn't exist
  4. [DELETE] /api/Inventory/{label} - Removes the item with the specified label from the inventory
  5. [PUT] /api/Inventory/Expire/{label} - Immediately expires the item with the specified label.

json format: { "Label": "Foo", "Expiration": "2016-12-31T07:00:00Z", "Type": "Metal" }
